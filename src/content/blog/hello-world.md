# Hello World!

Chào bạn! Đây là bài viết đầu tiên trên hệ thống Blog mới của mình. Bài viết này được gõ hoàn toàn bằng Markdown trên **Obsidian** và tự động xuất bản lên web nhờ GitHub Actions.

## Tại sao lại dùng Markdown?

Việc sử dụng Markdown có những lợi ích cực kỳ lớn:
1. **Nhanh:** Gõ văn bản cực kỳ nhanh mà không cần rời tay khỏi bàn phím.
2. **Lưu trữ cục bộ:** Các file `.md` nằm hoàn toàn trong máy tính của mình.
3. **Mã nguồn mở:** Không sợ nền tảng khác sập hay thu phí.

Dưới đây là một đoạn code để demo:

```javascript
function sayHello() {
  console.log("Hello từ Obsidian!");
}
sayHello();
```

> "Một khi bạn đã quen với Markdown, bạn sẽ không muốn dùng Word nữa đâu."

Bạn có thể chèn hình ảnh, tạo bảng, và làm mọi thứ. Hệ thống sẽ tự động chuyển nó thành một trang web tuyệt đẹp! 🚀
